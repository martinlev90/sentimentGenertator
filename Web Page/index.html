Conclusion

-> The graph of success vs rank shows a pattern of a known probability distribution:
the Gaussian distribution.
Something that we can expect because normal distribution is very common in the nature.
for more details on it visit the wiki page

What we can conclude from this is that a modarate sentiment will produce better success than the clear sad or happy sentiment.


-> On the graph of average rank by year we got that as time pass more and more happy songs gain high rank.
Something that we can think that represents the reality: in the 70s and 80s the songs were more happy than previeous years.
Later years (90s) the boy bands and pop music was popular and in the 2000s hip-hop and club music became popular.
Still in the recent years there is decrease in sentiment(rise of new hip hop like Drake and acoustic music with more sad narative).

-> In the graph of top 20 artists we see artists from pop and R&B more seccesfull, artists with strong sentiment in the music as well the lyrics (Michael Jackson, Mariah Carey, Elton Jhon, Eretha Franklyn).
We have a suggestion that the sentiment of the voice and singging also play an important role in the success of the song.

-> Avrage sentiment by decade shows also the Gaussian distribution, and also shows that modarate sentiment will cause better success in the billboard rank.

The Algorithm

-> We processed the data set using language analysis tools, We processed the data set using language analysis tools,
which can take raw English language text input (Stanford nlp java libraries)

-> For each artist and for each record we did a statistical analysis based on our scale

-> Over this analysis we calculated for each artist the average sentiment and his success on billboard ranking

-> Main problem – this is a computationally heavy algorithm, for a big dataset it takes a lot of time to process!

-> Solution – split the dataset to small chunks and reduce.

HOW IT WORKS

-> The algorithm uses Deep Learning for Sentiment Analysis

-> The model builds up a representation of whole sentences based on the sentence structure.
It computes the sentiment based on how words compose the meaning of longer phrases.
This way, the model is not as easily fooled as previous models (based on word isolation).

-> The underlying technology of this model is based on a new type of Recursive Neural Network that builds on top of grammatical structures.

-> A paper can be found here: Recursive Deep Models for Semantic Compositionality Over a Sentiment Treebank
